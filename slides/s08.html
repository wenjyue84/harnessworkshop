Browser Skills Breakdown

Two different browser automation skills for two completely different jobs. Know when to use which.

High-Level Research Tool

🌐 Agent Browser

Semantic web navigation. Understands pages like a human — click, scroll, extract meaning.

Under the Hood
Headless Playwright / Puppeteer automation
Best For
Research, scraping docs, data extraction, form automation

Low-Level Debugging Tool

🔍 Dev Browser

DOM-level debugging. Inspects console, network, memory — like having DevTools open programmatically.

Under the Hood
Chrome DevTools Protocol (CDP)
Best For
Debugging local apps, reading console errors, testing UI interactions
