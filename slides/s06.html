🚫 The Problem Context7 Solves

LLMs generate code using training data that is 1–2 years old. They fabricate APIs that no longer exist, use deprecated patterns, and give generic answers that don't match your actual package version. Context7 fixes this at query time.

💡 Feynman Explanation

Imagine asking a chef to cook a dish, but their recipe book is 2 years old. Some ingredients have changed, some techniques are discontinued. Context7 is like handing the chef today's menu from the restaurant, right before they cook.

🔎

resolve-library-id

Converts a library name (e.g. "Next.js") into a Context7-compatible ID for precise doc lookup.

📄

query-docs

Fetches current, version-specific docs for that library and injects them directly into your prompt's context.

🔧 Install in Claude Code

# Option 1: Auto setup (recommended)
npx ctx7 setup

# Option 2: Manual (local)
claude mcp add --scope user context7 --
  npx -y @upstash/context7-mcp --api-key YOUR_KEY

# Option 3: Remote (HTTP transport)
claude mcp add --scope user
  --header "CONTEXT7_API_KEY: YOUR_KEY"
  --transport http context7
  https://mcp.context7.com/mcp
