PRD Skill & Q&A

Turn a plain-English product idea into a machine-readable JSON specification — in one command.

1

Launch the PRD Skill

Type the slash command in Claude Code. The skill loads a specialized PRD-writing system prompt.

/prd
2

Describe your feature

Claude interviews you with structured questions about users, goals, constraints, and success metrics.

e.g. "Build a booking system for Pelangi Hostel"
3

PRD generated as Markdown

Claude outputs a structured prd.md with goals, user stories, acceptance criteria, and tech constraints.

→ prd.md saved
4

Convert to JSON

Ask Claude: "Convert prd.md to prd.json with structured fields." The JSON version is machine-readable — agents can load it directly as context.

→ prd.json ready for agents
5

Feed JSON to the Manager Agent

The Manager Agent reads prd.json and auto-generates task tickets for Coder, Tester, and Reviewer agents.

→ autonomous execution begins
📄 prd.md (excerpt)
## Booking System — Pelangi Hostel
### Goal
Enable guests to book rooms online
without phone calls or manual forms.
### User Stories
- As a guest, I can select dates & room
- As staff, I can view today's arrivals
### Acceptance Criteria
- Booking confirmed within 3 seconds
- Payment via Stripe or FPX
Tech: Next.js, Prisma ORM, Stripe
