claude-mem — Persistent AI Memory

Claude forgets everything when a session ends. claude-mem is a Claude Code plugin that automatically captures what Claude does across sessions, compresses it (~10× token reduction), and injects relevant context back at session start.

🧠
The Auto-Indexing Assistant

Imagine an assistant who listens to every meeting, extracts the key decisions and facts into a searchable index, and reads that index before tomorrow's meeting. You don't have to repeat yourself — they already know your history.

claude-mem works passively in the background. Every tool call, decision, and significant output gets compressed into a semantic observation and stored. The next session, relevant memories surface automatically.

⚙️ How claude-mem works

🔴 Capture

📡 Worker service (port 37777)
🪝 PostToolUse hook fires
📝 Raw output captured (~5k tok)
🤖 AI compresses to ~500 tok

🟢 Recall

🔍 search — full-text + filter
📅 timeline — chronological context
📄 get_observations — fetch details
⚡ 10× token efficient vs raw
Storage: SQLite (session data) + Chroma vector DB (semantic search)
Session 1 — captured facts
Stack: Next.js + Prisma DB: Neon PostgreSQL Hosting: Vercel + Railway Decision: REST → tRPC Bug: cascade delete issue in user model Pattern: always wrap API calls in try-catch
↓ 10× compressed ↓
Session 2 — context loaded
Claude already knows your stack, decisions, and patterns.
⚖️
claude-mem vs MEMORY.md

📝 Manual MEMORY.md

  • You write it manually
  • Only what you remember to add
  • Full content on every load
  • Not searchable
  • Full editorial control

🧠 claude-mem Plugin

  • Automatic capture (passive)
  • Everything Claude does
  • 10× compressed + search-retrieved
  • Vector + keyword searchable
  • AI decides what to compress
ℹ
Use both: claude-mem for automatic recall, MEMORY.md for facts you explicitly want Claude to always know (architectural decisions, personal preferences).

Hands-On Exercise

01

Install the plugin

In Claude Code, run: /plugin marketplace add thedotmack/claude-mem then /plugin install claude-mem. It will auto-install dependencies.

02

Work for 10 minutes

Do any coding task with Claude. Ask it to create files, analyze code, run commands. claude-mem is silently capturing everything.

03

Test recall in a new session

Start a fresh Claude Code session and ask: "Search your memory for what we did earlier" or "What do you know about this project?"

04

Browse the web UI

Open http://localhost:37777 in your browser to see all captured memories with search and filtering.
