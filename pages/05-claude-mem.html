claude-mem

Claude forgets everything when a session ends. claude-mem is a Claude Code plugin that captures facts, decisions, and patterns you want to persist — so Claude remembers across sessions by loading them into context.

The Smart Meeting Notebook

Imagine if your notebook automatically extracted the most important facts from every client meeting and organized them into a searchable index — and the next morning, your developer read that index before starting the day. That is claude-mem: auto-capture plus persistent recall across all Prisma projects.

Without claude-mem, every session starts from zero. Claude has no memory that Federation Oil wants data exported as CSV, that Rui Sin Plastic uses shift-based monitoring, or that the Pelangi chatbot needs to respond in three languages. With claude-mem, those facts live in MEMORY.md — a structured file that loads into every session automatically, giving Claude your accumulated client and project knowledge from day one.

ℹ
claude-mem is passive. It captures automatically during normal use. You can also edit MEMORY.md directly to add, correct, or delete facts.

Hands-On Exercise

Exercise 05 — Set Up Persistent Memory8 min
1
Check if claude-mem is installed
Look in Claude Code settings under Plugins. If not installed, check the Anthropic docs for the current installation method.
2
Find your MEMORY.md
Typically at: ~/.claude/projects/[project-hash]/memory/MEMORY.md. You can also ask Claude: "where is my MEMORY.md?"
3
Add 3-5 Prisma-specific facts manually
Open MEMORY.md and add known facts: client preferences (e.g., "Federation Oil wants CSV exports"), tech decisions (e.g., "Supabase for chatbot session state"), and project constraints (e.g., "LHDN CP21 form schema must not be modified").
4
Start a new Claude session
Close and reopen Claude Code. Ask: "What do you know about this project?" Claude should reference your MEMORY.md facts.
5
Correct any wrong memories
If Claude recalls something incorrectly, edit MEMORY.md directly. Remove wrong facts and replace with accurate ones.

Key Takeaways

📷
claude-mem is passive — it captures automatically. No commands needed. It listens during normal conversations and extracts important facts.
📄
MEMORY.md is your source of truth. Edit it directly to correct wrong memories, add important context, or remove outdated facts.
⚡
Sessions start with MEMORY.md pre-loaded. Claude enters every conversation already knowing your project history, decisions, and preferences.
