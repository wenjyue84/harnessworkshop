Agent Browser Skill

Semantic Web Navigation — giving Claude live eyes on the internet.

The agent-browser skill gives Claude the ability to open URLs, navigate pages, fill forms, click elements, and extract content — using a headless Chrome browser. You trigger it by installing the skill and saying browse to X and get Y.

💡
The Mental Model
Without the browser skill, Claude is a brilliant person locked in a room with no windows — it can reason, but cannot see the web. The agent-browser skill opens that window. Now Claude can read any live webpage, not just what you paste.

Open URLs

Multi-page Flow
Follow links, paginate through results, crawl across multiple pages.

Where Skills Live
Skills are Markdown or YAML files that inject system-level instructions into Claude context. Canonical location: ~/.ai/Skills/agent-browser/. Claude Code also checks ~/.claude/skills/ via a directory junction.

🏋 Exercise

Hands-On: Use the Agent Browser
1
Locate the skill
Run ls ~/.ai/Skills/. Confirm an agent-browser/ directory exists. If missing, clone it from your skills repository.
2
First browse request
In Claude Code: Use the browser to go to example.com and tell me what the page is about. Watch Claude activate the skill and return live content.
3
Observe the flow
Watch Claude emit step-by-step status messages: loading skill → launching Chrome → navigating → extracting. This maps to the animation above.
4
Extract structured data
Try: Go to docs.anthropic.com and extract all headings from the sidebar navigation. Expect a clean list, not raw HTML.
5
Form interaction
Try: Navigate to duckduckgo.com, search for Claude Code hooks, and return the first 3 result titles and URLs.
⚠
Know the Limits
Cannot bypass CAPTCHA or SMS 2FA. Does not work behind corporate VPNs without Chrome proxy config. For authenticated resources, use GitHub MCP or AWS CLI instead.

🎯 Key Takeaways

What to Remember

🌐
Browser skill = live web access. Claude reads any URL, not just pasted content.
⚙
Works with JavaScript-rendered pages. Headless Chrome executes JS fully.
📊
Returns structured data. Claude parses the DOM into clean text, not raw HTML.
🤖
Fully autonomous. Forms, clicks, navigation — no manual steps required.
🔗
Composable. Chain with file writes, GitHub MCP, or AWS CLI for full automation pipelines.
