# Option B: Download from nodejs.org

Hands-On Exercise

Exercise 01 — Install & Authenticate5 min
1
Subscribe to Claude Pro
Go to claude.ai → click Upgrade → choose Pro at $20/month → confirm payment.
2
Install Node.js
winget install OpenJS.NodeJS in PowerShell (as Administrator), or download from nodejs.org. Verify: node --version
3
Install Claude Code globally
npm install -g @anthropic-ai/claude-code — installs the claude binary system-wide.
4
Verify the install
Run claude --version. Should show a version number. If command not found, restart terminal and confirm npm global bin is in your PATH.
5
Authenticate via OAuth
Run claude with no arguments. Browser opens for OAuth. Log in with your Claude Pro account. Return to terminal — you will see a Ready confirmation.
⚠
Windows users: Run PowerShell as Administrator for npm global installs. On permission errors: npm config set prefix "%APPDATA% pm" then retry.

Claude.ai vs Claude Code

Claude.ai (Web App)
  • Chat interface in the browser
  • Great for Q&A, writing, analysis
  • Cannot read your local filesystem
  • Cannot run terminal commands
  • Conversations are largely isolated
Claude Code (CLI)
  • Terminal-native, runs in your project
  • Reads, writes, edits your actual files
  • Executes shell commands autonomously
  • Loads CLAUDE.md for persistent context
  • Supports hooks, plugins, automation
