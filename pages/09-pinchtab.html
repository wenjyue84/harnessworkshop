Pinchtab

Browser Tab Context Capture — one shortcut, full page awareness.

Pinchtab is a browser extension that captures your currently active browser tab — the URL, page title, selected text, and full page content — and injects it as context into Claude Code. Instead of copy-pasting, you pinch the tab and Claude instantly knows what you are looking at.

💡
The Mental Model
Imagine having a colleague who can glance at exactly what is on your screen and immediately understand it — without you having to explain or copy anything. Pinchtab is that glance: one shortcut, and Claude gets full context of your browser tab.

⚡ How It Works

Press the Pinchtab shortcut and watch page content flow from your browser into Claude Code context. Select some text first to give Claude a focused injection point.

📋 What Gets Captured

URL
The full URL of the active tab, including query params and fragments.
Title
The page title, used by Claude to understand the document context.
Selected Text
Any text you highlight before pressing the shortcut gets priority injection — Claude focuses on that first.
Full Content
The visible page text, cleaned of navigation chrome and ads.
Works With
Any browser tab, including JavaScript-rendered apps. Also works with PDFs open in the browser.

🏋 Exercise

Hands-On: Install and Use Pinchtab
1
Install the extension
Search for Pinchtab in the Chrome Web Store or Firefox Add-ons. Install and pin it to your toolbar.
2
Configure Claude Code integration
Follow the extension setup wizard. It will ask for the Claude Code endpoint URL and shortcut key preference (default: Ctrl+Shift+P on Windows, Cmd+Shift+P on Mac).
3
Open any webpage
Navigate to any page you want to discuss with Claude — docs, a bug report, a competitor page, a Stack Overflow answer.
4
Highlight text
Select a specific paragraph or code block on the page. This becomes the priority content Claude focuses on first.
5
Press the shortcut
Hit Ctrl+Shift+P (or your configured shortcut). Switch to Claude Code and ask a question about the page — Claude already has full context.
📚
Power Combo
Use Pinchtab to capture a page, then ask Claude to use the agent-browser skill to follow links from that page. You get human-guided starting point + autonomous deep crawl.
