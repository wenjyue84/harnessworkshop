Firecrawl MCP

Firecrawl converts any URL to clean, LLM-ready markdown. The MCP server adds scraping, crawling, search, and structured extraction directly to Claude — 11 tools for web intelligence.

🔥
The Paper Mill

The web is 99% HTML noise: ads, navigation bars, cookie banners, tracking scripts, sidebars, footers. Dumping raw HTML into a context window is wasteful — most of it is irrelevant.

Firecrawl is the paper mill. A messy forest of HTML goes in; clean, dry markdown comes out. One URL in, structured content that fits your context window out. 45,000 characters of HTML → 3,200 characters of clean markdown.

11 MCP Tools
firecrawl_scrape
Single URL → clean markdown/HTML/JSON
firecrawl_search
Web search with optional result scraping
firecrawl_crawl
Async multi-page crawl with depth control
firecrawl_map
Discover all indexed URLs on a site
firecrawl_extract
Structured data extraction with custom schema
firecrawl_agent
Autonomous research agent — browses independently

Plus: check_crawl_status, browser_create/execute/delete/list for persistent browser sessions.

⚡
One-Command Install
# Easiest way — Claude Code CLI installer: claude mcp add firecrawl -e FIRECRAWL_API_KEY=your-api-key -- npx -y firecrawl-mcp # Or manually in ~/.claude/.mcp.json: { "mcpServers": { "firecrawl": { "command": "npx", "args": ["-y", "firecrawl-mcp"], "env": { "FIRECRAWL_API_KEY": "fc-your-key" } } } }

Get a free API key at firecrawl.dev/app/api-keys. Free tier is generous for learning.

Hands-On Exercise

01

Get your API key

02

Install Firecrawl MCP

Run the one-liner: claude mcp add firecrawl -e FIRECRAWL_API_KEY=fc-your-key -- npx -y firecrawl-mcp

03

Scrape a documentation page

Ask Claude: "Scrape docs.anthropic.com/claude/docs/tool-use and extract the key concepts"

04

Do competitive research
