Firecrawl MCP

Web Scraping for AI Agents — any URL in, clean markdown out.

Firecrawl is a web scraping API that converts any URL into clean, LLM-optimised markdown. For Prisma Technology Solution, this means Claude can scrape the LHDN website for the latest PCB/EPF/SOCSO rate updates, crawl competitor chatbot documentation, and pull client portal content directly into context — without manual copy-pasting.

💡
The Mental Model
The web is 99% HTML noise: ads, navbars, cookie banners, JavaScript widgets. Firecrawl is the paper mill that takes a messy forest and outputs clean, dry paper — exactly the kind of content Claude can actually process. One URL in, clean markdown out. Critical for keeping Prisma's payroll software up-to-date with LHDN regulation changes.

🔧 Firecrawl Endpoints

/scrape
Single Page
One URL to clean markdown. Executes JS.
/crawl
Entire Site
Recursively crawl all linked pages.
/search
Web Search
Search query → results as markdown.
/extract
Structured Data
Extract typed data with a JSON schema.
/map
Site Map
Get all URLs from a site quickly.

🎯 Key Takeaways

What to Remember

🖹️
Firecrawl = clean web content for Claude. HTML noise stripped, markdown returned. Critical for LHDN regulation scraping.
🔑
API key required. Free tier available at firecrawl.dev — sufficient for Prisma's research use.
📄
Returns markdown, not raw HTML. Claude reads markdown natively; HTML wastes context tokens when building RAG knowledge bases.
🌐
Use /scrape for LHDN pages, /crawl for client portals. Mix endpoints for payroll research and chatbot knowledge base building.
