Installation and Usage

terminal
# Install Gemini CLI:
$ npm install -g @google/gemini-cli

# Or use npx (no install needed):
$ npx @google/gemini-cli

# Get free API key at aistudio.google.com:
$ export GOOGLE_API_KEY=your_key_here

# Basic usage on Prisma's IoT codebase:
$ gemini "Explain the sensor data flow in prisma-iot-counter"
$ gemini "Write JSDoc for all functions in src/sensors/ultrasonic.js"

# YOLO mode — bulk add JSDoc to all 50 IoT sensor files for free:
$ gemini --yolo "Add JSDoc comments to all .js files in src/sensors/"
Running autonomously across all sensor files... (free tier, no cost)
⚠️
Same safety rules apply as dangerously-skip-permissions
--yolo executes file writes, bash commands, and file deletions without asking. Define scope tightly. Use on trusted projects only. Always review output.
