Gemini --yolo

Gemini CLI is Google's free, open-source AI coding agent with a --yolo flag for full autonomous mode. With Google AI Studio's free tier, it's a powerful, cost-free complement to Claude Code.

🐎
The Reliable Workhorse

If Claude Code is the thoroughbred — premium, trained for performance, built for precision work — Gemini CLI with --yolo is the reliable workhorse: free, tireless, good enough for most everyday tasks, and you can run it all day without watching the API meter.

Different tools, same stable. Senior AI engineers use both: Claude for complex autonomous work, Gemini for bulk tasks where cost or rate limits matter.

--yolo vs other autonomous modes
# Gemini --yolo (per session) gemini --yolo "refactor all files in src/" gemini -y "refactor all files in src/" # short form # Ctrl+Y toggles yolo mode mid-session (no restart needed) # GEMINI_YOLO_MODE=true persists for the session export GEMINI_YOLO_MODE=true # vs Claude Code (per session) claude --dangerously-skip-permissions # vs Claude Code (persistent in settings.json) { "permissions": { "defaultMode": "bypassPermissions" } }
⚠
Same safety rules apply as Claude's bypass mode. --yolo executes file writes, bash commands, and deletions without confirmation. Use in isolated environments on well-scoped tasks only.

Hands-On Exercise

01

Get your free API key

Go to aistudio.google.com → "Get API key" → "Create API key". The free tier gives generous usage for learning and development.

02

Install and test

Run: npm install -g @google/gemini-cli && export GOOGLE_API_KEY=your_key && gemini "Hello"

03

Try --yolo on a safe task

In a test directory: gemini --yolo "Add a README.md summarizing this project". Observe the autonomous behavior — no confirmations.

04

Compare quality vs Claude Code

Give the same task to both Claude Code and Gemini CLI. Compare output quality, approach, and style. Note where each excels — use this to decide which to use for future tasks.
