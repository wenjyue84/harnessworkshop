Safety: Principle of Least Privilege
Claude will use whatever permissions your CLI credentials have. Always use dedicated IAM roles with only the permissions needed. Never configure Claude Code with root credentials or admin access. Review generated commands before allowing execution on production accounts.

🏋 Exercise

Hands-On: Configure AWS CLI with Claude
1
Install AWS CLI
Windows: winget install Amazon.AWSCLI or download from aws.amazon.com/cli. Mac: brew install awscli.
2
Configure credentials
Run aws configure. Enter your Access Key ID, Secret, default region (ap-southeast-1 for Singapore), and output format (json).
3
Verify identity
Run aws sts get-caller-identity. Should return your Account ID and User ARN. If it fails, check your credentials.
4
First Claude query
In Claude Code, ask: List all my S3 buckets. Claude will run aws s3 ls and present the results in readable form.
5
Log query
Ask: Show me CloudWatch logs for [your service name] from the last hour. Claude will find the log group and fetch recent entries.
6
Advanced
Ask Claude to help set up a new EC2 instance with your preferred AMI, instance type, and security group. Review the generated commands before running them.
