GitHub MCP

AI-Native Git Operations — issues, PRs, and reviews in one continuous flow.

The GitHub MCP server connects Claude directly to the GitHub API. For Prisma Technology Solution, this means Claude can read IoT bug reports, create pull requests for chatbot knowledge base updates, manage repos like prisma-iot-counter and ai-livechat — all without leaving the terminal or manually switching to the browser.

💡
The Mental Model
Normal GitHub workflow: you switch between editor, browser, and terminal to read issues, write code, open PRs, and respond to reviews. With GitHub MCP, Claude is simultaneously the developer AND the project manager — reading the IoT sensor bug, writing the fix, opening the PR, and writing the review in one continuous flow.

🏋 Exercise

Hands-On: Connect GitHub MCP for Prisma Repos
1
Create a Personal Access Token
Go to github.com → Settings → Developer settings → Personal access tokens → Generate new token (classic). Select scopes: repo, issues, pull_requests. Copy the token — it is shown only once.
2
Add to .mcp.json
Open ~/.claude/.mcp.json and add the github server config shown above. Replace ghp_your_token_here with your actual token.
3
Restart and verify
Restart Claude Code. Run claude mcp list — you should see github (connected).
4
List Prisma open issues
Ask Claude: List open issues in prisma-iot-counter. Claude should return a formatted list of sensor bugs and backlog items.
5
Create an IoT bug issue
Ask: Create a GitHub issue titled 'Ultrasonic sensor reading intermittent on injection machine line 3' in prisma-iot-counter. Verify it appears on GitHub.
6
Advanced: PR for chatbot RAG update
Ask Claude to review the latest pull request in ai-livechat (your RAG chatbot repo) and summarise what knowledge base entries were added or changed.
⚠
PAT Security
Your Personal Access Token grants write access to Prisma's repositories. Store it only in .mcp.json (not committed to git). Use fine-grained PATs where possible to limit scope to specific repos like prisma-iot-counter, ai-livechat, and prisma-payroll.
